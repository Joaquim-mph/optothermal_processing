# Notes Directory

This directory contains **project management and internal development documentation** that is not intended for end users.

## Directory Structure

### `planning/`
Project planning documents, architecture proposals, and upgrade plans.

**Contents:**
- `TUI_V3_UPGRADE_PLAN.md` - TUI version 3.0 upgrade planning document

### `progress/`
Progress reports and status updates for ongoing development work.

**Contents:**
- `TUI_V3_PROGRESS_REPORT.md` - TUI v3.0 upgrade progress tracking

### `implementation/`
Detailed implementation guides for developers working on specific features.

**Contents:**
- `matplotlib_plot_display_implementation_guide.md` - Textual TUI matplotlib integration guide

### `features/`
Feature completion documents and implementation status reports.

**Contents:**
- `ITS_PHOTORESPONSE_FEATURE.md` - ITS photoresponse feature completion status

## Purpose

These documents are:
- ✅ Internal development documentation
- ✅ Project planning and tracking
- ✅ Implementation guides for contributors
- ✅ Historical records of feature development

These documents are **NOT**:
- ❌ User-facing how-to guides (see `docs/guides/`)
- ❌ Reference documentation (see `docs/reference/`)
- ❌ Architecture documentation (see `docs/architecture/`)

## For End Users

If you're looking for how to **use** the software, please see:
- **Main documentation**: `/docs/`
- **Quick start**: `README.md`
